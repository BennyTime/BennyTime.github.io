# The Grand Divide

A comprehensive documentation site for **The Grand Divide** D&D campaign setting, built with [Docusaurus](https://docusaurus.io/).

## About

**The Grand Divide** is a D&D campaign setting in the world of Vaeloran—a realm shattered by the Shard War, where history has been destroyed and nine nations stand divided, each shaped by an Elemental Shard's power.

This site contains:
- **Timeline** - The documented history from the Early Eras through the Current Era (Year 1109)
- **World Regions** - Detailed information about all 11 nations and territories
- **Gods & Religion** - The Creators, Divinities, and spiritual traditions
- **General Knowledge** - Laws, widely-known facts, legends, and practical information
- **Organizations & Guilds** - Major factions, guilds, and ruling bodies across Vaeloran
- **Races** - 16 homebrew playable races with full mechanics
- **Languages & Realms** - Language families and planar cosmology
- **Items & Equipment** - New weapons, firearms, and updated gear
- **Campaign Log** - Session notes and campaign updates (coming soon)

## Local Development

```bash
cd grand-divide
npm install
npm start
```

This command starts a local development server at `http://localhost:3000/grand-divide/`.

## Build

```bash
npm run build
```

This command generates static content into the `build` directory for deployment.

## Deployment

The site is automatically deployed to GitHub Pages when changes are pushed to the main branch.
